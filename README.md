# flycheck-tcl

[![MELPA][badge-melpa]](http://melpa.org/#/flycheck-tcl)

A flycheck checker for Tcl using ActiveState's tclchecker

As usual, from [MELPA][].

Installation
------------

In your `init.el`:

```cl
(with-eval-after-load 'flycheck
  (flycheck-tcl-setup))
```

[MELPA]: http://melpa.org
